The process of assigning predefined categories or labels to text data. It is a core task in [[Natural Language Processing]] (NLP) used in applications like **spam detection, sentiment analysis, topic labeling, news categorization, intent detection** and more. 

Text classification takes **raw textual data** and transforms it into structured information by predicting which class the input text belongs to. The general workflow involves 

Text Preprocessing $\rightarrow$ Feature Extraction / Representation $\rightarrow$ Model Building $\rightarrow$ Prediction and Evaluation.
